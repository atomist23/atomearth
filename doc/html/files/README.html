atomearth — A pcap to Google Earth feed generator

This application will generate a KML feed for use with Google Earth from network traffic. It can parse a pcap file or listen to ethernet traffic in realtime.

Installing

Install manual 

  gem install atomearth

Configuration 

  Run atomearth as root to copy the configuration files to /etc/atomearth/config.yaml

    sudo atomearth

  Then edit the configuration file for your needs.

Running 

  atomearth start - run atomearth in background (daemonize)
  atomearth run   - run atomearth in foreground
  atomearth stop  - stop atomearth daemon
